# Django ecommerce

This is an example of Django ecommerce

## How to install

```
make install
```

## How to run

```
make run
```

## Makemigrations

```
make mm
```

## Migrate
```
make m
```

## Remove (clean) venv
```
make clean
```

## How to use

Start a development server at http://127.0.0.1:8000/ using `make run` command.
